# pKit

pKit, also called phpKit, is a small library for PHP projects. 

You can find its documentation [here](http://iexit1337.github.io/pkit/index.html).

It includes

  - M(odel) - V(iew) - C(ontroller) - System
  - Routing
  - Sessions
  - Password Hashing

Planned stuff:

- JSON Helpers
- Language System
- Plugin System
